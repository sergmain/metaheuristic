standalone - application is running without needing to configure processors and dispatcher, front-end must communicate with server via port 64967
disable-check-frontend - is app in standalone mode then it'll check presence of front-end, if front-end not presented, then server will be shut down.
disk-storage - variables and other blobs assets are stored on disk as a files, instead of database as BLOBs
websocket - enable communication between dispatcher and processors via WebSocket
